TERCEIRO SEMESTRE:

Acessibilidade

No terceiro semestre do curso de Sistemas para Internet na disciplina de Acessibilidade foi proposto aos alunos a criação de páginas web conforme as diretrizes de acessibilidade WCAG (diretrizes de acessibilidade para conteúdo web). Este site foi desenvolvido conforme as normas da cartilha W3c, cada parte do site foi estudado e aplicado conforme as normas estabelecidas, tecnologias utilizadas HTML5, CSS3, Java Script, Bootstrap e jQUERY. O Bootstrap é um framework mais atual de código de fonte aberto (Open Source), para desenvolvimento de componentes de interface e Front End. Este é um sítio responsivo, com menu lateral e bem interativo ao usuário. Este site está em sua forma original e não foi alterado para o público daltônico.
