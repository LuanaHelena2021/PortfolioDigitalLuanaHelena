SEGUNDO SEMESTRE:


Padrões de Projetos de SÍtios Internet II.


No segundo semestre do curso de Sistemas para Internet na disciplina de Padrões de Projetos de Sítios Internet II foi proposto aos alunos o desenvolvimento de uma página web em HTML, CSS, Java Script(JS), Materilize e jQuery. Os alunos foram avaliados pela criatividade e entendimento na utilização das tecnologias e suas funções. O Materialize é um framework front-end responsivo, nele encontramos códigos como botões, menu, ícones, rodapé e alguns templates de sitios prontos para a utilização, o maior desafio para se trabalhar com o materialize é encontrar no estilo (CSS) ou no Java Script (Js) os elementos corretos a serem modificados, Como se trata de templates prontos todo o código já está pronto, este foi o segundo sitio para avaliação. Este site está em sua forma original e não foi alterado para o público daltônico. "TUDO É PERSISTÊNCIA E MUITO TREINO!
