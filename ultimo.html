SEXTO SEMESTRE:

TÓPICOS ESPECIAIS III.

No sexto semestre de 2022 do curso de Sistemas para Internet na disciplina de Tópicos Especiais III, foi desenvolvido um trabalho para apresentação individual dos comandos básicos do GIT. Conforme o site de documentação do Github, o Git é um controle de versionamento de arquivos, monitora o histórico de alterações conforme as equipes colaboram com os projetos, conseguem recuperar as versões anteriores, consegue revisar o que foi alterado e por quem foi alterado. O Git Bash é o primeiro passo para se trabalhar em seu Windows, é um aplicativo para ambientes do Microsoft Windows que oferece a camada de emulação, o prompt de comando é parecido com o prompt de comando normal do computador, o que diferencia é a logomarca do git bash e as letras coloridas. Os commits são descrições do que já foi modificado. Quando se olha um commit podemos verificar os arquivos modificados individualmente, sua base de código e todo seu histórico, os repositórios podem ser alterados também no vscode, exatamente no console conforme executamos no git bash e ambos podem ser compartilhados no Github, assim como podemos clonar os repositórios. O Github é uma plataforma de hospedagens de arquivos com Open Source (códigos abertos) para serem compartilhados em forma de repositórios que permitem armazenar e contribuir no código de projetos. O Visual Studio Code (VSCODE) é uma ferramenta importantíssima para os desenvolvedores quando falamos em git, o vscode fornece suporte abrangente para a edição de código, navegação e compreensão juntamente com uma depuração leve, todos os códigos utilizados no git bash podem ser utilizados no console do vscode, inclusive clonar repositórios diretos do github, significa fazer cópias direto dos arquivos do github.
