PRIMEIRO SEMESTRE:

Padrões de Projetos de SÍtios Internet I.

No primeiro Semestre de 2020 foi desenvolvido, na Disciplina de Padrões de Projetos de Sítios de Internet I, páginas web e elementos de design usando HTML e CSS. Apesar de um site simples è um cardápio de uma cafeteria, o trabalho foi individual e cheio de alinhamentos de imagens, uma junção de várias páginas, harmonização de cores, alinhamento de textos e rodapé, este foi o primeiro site desenvolvido. Neste site o Professor e Mestre Felipe Maciel nos ensinou float: left ( alinhamento a esquerda), float:right(alinhamento a direita), width(a largura do elemento), height(a altura). No menu temos os links para as outras páginas mostrando os outros produtos do cardápio. E no rodapé um alinhamento central com os links para as mídias sociais. Este site está em sua forma original, não foi alterado para o público daltônico.
